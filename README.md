# ottawa transport widget

an ios home screen widget that provides real-time bus departure information for ottawa's oc transpo system. it allows users to pin their favorite stops and routes for quick access to live transit data.

*   **smart prediction**: automatically toggles between inbound and outbound trips based on the time of day.
*   **location-aware**: highlights stops near you so you always know which one to walk to.
*   **customizable**: choose specific routes and stops to track.
*   **native performance**: built with swift functionality for a seamless ios experience.
*   **privacy-focused**: direct communication with transit apis; no personal data collection.
